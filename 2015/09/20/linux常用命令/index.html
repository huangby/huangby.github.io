Linux常用命令

  • Linux
Linux

一、软连接：
ln -s 源文件 目标文件
其类型为：lwrxwrxwrx（所有的人都可以访问，但是最终取决于目标文件）
等同于window下的快捷方式

二、硬连接=cp -p+同步更新
ln 源文件 目标文件
其类型为：-wrx…….(同源文件)
其特点是：和源文件同步，（时间，大小，权限等等）[类似于 cp -p命令,需要加入-p 选项才能保证拷贝的时间一样]
但是又不一样的是，硬连接是实时同步的。

删除源文件后，软链接就不起作用了，但是硬链接可以使用，因此可以使用硬连接进行文件实时备份

每个文件必须有一个inode节点（数字标识） 使用命令 ls -i查看文件的inode节点
硬链接和源文件有相同的inode节点
一个inode节点可以映射多个文件
软连接可以夸分区，但是硬链接不能夸分区。

三、chmod 改权限

代表字符 权限 对文件含义 对目录含义

r 读权限 可以查看文件内容 可以列出目录中的内容

w 写权限 可以修改文件的内容 可以在目录中创建/删除文件

x 执行权限 可以执行文件 可以进入目录

也就是说，一个文件(不是目录)，就算自己没有执行权限，也可以删除该文件，只要有在此文件所在目录有可执行权限就行。
chown 改拥有者（另添加用户 1.useradd 用户名，2.passwd 用户名）
chgrp 更改所在组
umask (-S)查看缺省的权限,得出的如果是数字，表示的是权限掩码值，需要将777减去相应的值(第一个数字代表特殊权限位，后三个才是权限掩码)，虽然如此，但是linux里面有这样的规则，缺省的文件的权限都没有可执行的权限。因此就算umask得出的有可执行权限，但是新建的文件没有可执行权限。
umask 权限掩码值(=777-权限值) 这样就可以更改系统的缺省权限。

下面是有关查找的一些命令
四、which +命令 可以查找到命令所在路径
命令所在路径可以判断哪个命令能不能被使用。位于bin或者/usr/bin下的被所有用户使用。位于sbin下的只能被root使用。which能够查看到命令相关的别名。
whereis不仅仅可以找到路径，还能够提供命令帮助文件所在位置。
上述只能找到命令。

五、find查找文件所在路径。（建议不要在根目录下查找）
find [搜索路径] [搜索关键字] [连接符]
搜索关键字有下面的用法：
-name 根据文件名查找(该名为文件全名或者通配符)
-size 文件大小(以数据块block=512字节为单位)
【+2048：大于2048；-2048：小于2048；=2048等于2048】
-user 用户文件（查找隶属于所有者的文件）
-type 通过类型
-inum 通过查找inode节点
-ctime/atime/mtime(根据时间查找，以天为单位)【cmin/amin/mmin以分钟为单位】
其中c：表示change(改变-文件属性)
a：access（访问）
m：modify（修改-内容）
【-表示在…之内，+表示…超过】
find中的连接符1：-a 逻辑与，-o逻辑或
2：-exec 命令 { } \; {}表示find的查询结果，\表示转义符，使用命令的本身的含义
-ok 询问连接符
六、(linux特有命令)locate +文件名（在系统的文件数据库里面查找，所以很快，但是有时候会出现找不到的情况，因为存在的文件可能还没有及时更新到数据库里面，而find命令是在整个硬盘里面查找）
可以手动使用updatedb命令更新数据库。
查看文件内容—cat、more、tail、head、grep
七、grep +【指定的字符串】【源文件】
八、man 查看命令/配置文件的帮助文件(配置文件不能打路径，直接使用文件名)
man会优先查看命令的配置（1表示命令的帮助，5表示配置文件的帮助），当命令和配置文件重名，则 使用man 5 +配置文件名
九、whatis 获得索引的简单信息
命令+【–help】 可以获得命令的帮助
help命令可以获得内置命令。
十压缩解压—–gzip+文件名
linux的所有压缩格式在windows可以任意解压，但是想反则不是。
gizp压缩特点：
1、只能压缩文件。
2、不保留源文件。
gunzip/gzip -d 解压
十一、tar命令(.tar.gz)（打包压缩（解压解包）—gzip只是压缩/解压）
-c 产生.tar打包文件
-v显示详细信息
-f指定压缩后的文件名
-z打包同时压缩
-x解包.tar文件
tar [-zcvf] 包文件名 源文件名
十二、zip——-保留源文件
zip 压缩的文件名 源文件
zip -r 目录名 源目录文件名
十三、write +用户名【给用户发送消息】 【ctrl+D结束】
十四、wall 向所有人发送消息
wc -l 统计行数
快捷键
ctrl+l 清屏
ctrl+u 清楚当前行命令
history 查看之前的命令
十五、命令重命令
alias 别名=命令
unalias 删除别名
十六、输入/输出重定向（0：标准输出，1：标准输入，2标准错误输出）
>(输出，会被覆盖) >>(输出，可以追加) \<(输入) 2>(错误输出)
十七、将一个命令的输出传送给另外一个命令，作为他的输入（命令|命令|命令……）
