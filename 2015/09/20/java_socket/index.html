java socket总结

  • JAVA
JAVA

服务器端：
1.首先设置端口
ServerSocket ss=new ServerSocket(8888);
2.监听是否有客服端连接
Socket s=ss.accept();
这是一个阻塞函数，只有没有客服端连接时，一直处于当前位置。

3.获取输入流和输出流
BufferedReader br=new BufferedReader(
new InputStreamReader(
s.getInputStream()))；
BufferedWriter bw =new BufferedWriter(
new OutputStreamWriter(
s.getOutputStream()));
4.接下来就是循环接收和发送部分，注意上面的得到输入/输出流不能在循环里面。
使用br.readLine()，获取一行，注意的是，这里只是得到数据中的一行，如果数据里面有多行，可以多次调用该方法。
使用bw.write()写入数据。写入之后需要调用
bw.flush()方法，这样才能发送。(如果不调用该方法，接收端就不能立刻接收到该消息，要知道调用该方法)
这里还需注意的是，br.readLine()（实际上是s.getInputStream()方法）方法，在没有接收到数据时，会等待。
客户端：
1.设置需要连接的IP地址和端口
Socket s=new Socket(“127.0.0.1”,8888);
2..获取输入流和输出流(同服务器端)
3.接下来就是循环接收和发送部分，注意上面的得到输入/输出流也不能在循环里面。（同服务器端）
下面是个例子：
服务器：

1. public class server {
2. private BufferedWriter out=null;
3. private int i=0;
4. server() throws IOException{
5. ServerSocket s=new ServerSocket(8888);
6. System.out.println("正在监听.....................");
7. Socket socket=s.accept();
8. System.out.println("收到");
9. out=new BufferedWriter(
10. new OutputStreamWriter(
11. socket.getOutputStream()));
12. oper();
13. }
14. public void oper() throws IOException{
15. while(true){
16. out.write("this is no message\n" +
17. "so so so\n");
18. out.flush();
19. try {
20. Thread.sleep(1000);
21. } catch (InterruptedException e) {
22. // TODO Auto-generated catch block
23. e.printStackTrace();
24. }
25. }
26. }
27. public static void main(String[] args) throws IOException {
28. // TODO Auto-generated method stub
29. new server();
30. }
31. }

客户端：
